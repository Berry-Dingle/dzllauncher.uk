Features

Everything DZLL does today — and what’s coming next.

Core

  • Fast server browsing with search, favourites, sorting, and useful filters.
  • Blocklist support for suspicious servers (with clear join warnings).
  • Steam launch integration (Proton) with sensible launch options.

Mods

  • SteamCMD mod install / update / verify (optional; your choice).
  • Proton prefix watch-folder + preset workflow so mods actually load.
  • Manage installed mods UI (delete one / delete all safely).

Discord

  • Rich Presence with detail level: In Menus / In Game / On Server.
  • Privacy Mode hides server details even when “On Server” is selected.

Planned

  • Flatpak Steam support (experimental) — paths & sandbox considerations.
  • More launch QoL (start minimized, extra parameters presets, etc.).
