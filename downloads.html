Install notes

  • Steam (native) is the supported target right now.
  • DayZ runs under Proton; DZLL uses Steam to launch and connect.
  • Mod handling uses SteamCMD (optional) and a launcher watch-folder workflow.
Tip: If you want zero hassle, start with defaults and only enable auto-update/verify if you actually need it.
